## ADDED Requirements

### Requirement: 系统能够从设备拉取全屏截图并解码为位图
系统 SHALL 通过 AdvancedSharpAdbClient 底层 API 从已连接设备拉取全屏截图，并解码为跨平台可渲染的位图对象用于画布显示。

#### Scenario: 成功拉取并显示截图
- **WHEN** 用户点击“截图”按钮
- **THEN** 系统调用 `GetFrameBufferAsync()` 拉取图像数据
- **THEN** 解码后在画布中心显示截图
- **THEN** 状态栏显示原始尺寸

#### Scenario: 高分辨率设备降采样
- **WHEN** 设备分辨率超过渲染安全阈值
- **THEN** 系统自动降采样并保持宽高比
- **THEN** 状态栏显示已降采样信息

### Requirement: 系统能够支持交互式矩形裁剪
系统 SHALL 允许用户在画布上拖拽创建矩形裁剪区域，并支持拖拽顶点或边调整尺寸。

#### Scenario: 创建裁剪区域
- **WHEN** 用户在画布上按住鼠标左键并拖拽
- **THEN** 系统绘制实时跟随的矩形框
- **THEN** 释放鼠标后固定裁剪区域并显示 `(x, y, w, h)`

#### Scenario: Shift 锁定宽高比
- **WHEN** 用户按住 Shift 键并拖拽裁剪手柄
- **THEN** 系统锁定宽高比
- **THEN** 调整时保持初始比例

### Requirement: 系统能够提供像素坐标拾取与辅助叠加
系统 SHALL 在画布交互中提供像素坐标拾取、十字准线锁定和辅助网格显示。

#### Scenario: 鼠标悬停显示坐标
- **WHEN** 用户鼠标在画布上移动
- **THEN** 状态栏实时显示当前像素坐标
- **THEN** 坐标基于原始图像尺寸而不是缩放后视图尺寸

#### Scenario: 十字准线锁定
- **WHEN** 用户按下 Ctrl 键
- **THEN** 系统在当前光标位置绘制十字准线
- **THEN** 准线用于辅助像素对齐

### Requirement: 系统能够导出裁剪区域为独立 PNG 模板
系统 SHALL 将裁剪区域导出为独立 PNG 文件，并同时记录偏移量与原始尺寸元数据。

#### Scenario: 导出模板成功
- **WHEN** 用户点击“导出模板”按钮且裁剪区域有效
- **THEN** 系统导出 PNG 文件
- **THEN** 同时生成记录偏移量与原图尺寸的元数据文件

#### Scenario: 未创建裁剪区域时导出
- **WHEN** 用户未创建裁剪区域直接点击“导出模板”
- **THEN** 系统显示“请先创建裁剪区域”提示

### Requirement: 系统能够执行 OpenCV 模板匹配
系统 SHALL 使用跨平台 OpenCV 绑定执行 `TM_CCOEFF_NORMED` 模板匹配，并返回位置、尺寸、置信度与耗时。

#### Scenario: 成功匹配模板
- **WHEN** 用户加载模板并点击“匹配”按钮
- **THEN** 系统异步执行模板匹配
- **THEN** 画布绘制匹配结果框
- **THEN** 报告显示置信度与耗时

#### Scenario: 匹配低于阈值
- **WHEN** 最高置信度低于当前阈值
- **THEN** 系统显示“未找到匹配”提示
- **THEN** 日志输出最高置信度值

### Requirement: 系统能够异步处理图像操作避免 UI 阻塞
系统 SHALL 使用异步架构执行截图拉取、解码、裁剪、模板匹配与位图上传。

#### Scenario: 图像处理不阻塞 UI
- **WHEN** 系统执行耗时图像处理操作
- **THEN** UI 保持响应
- **THEN** 操作完成后自动刷新对应图层
